# Лабораторная работа №2: Очистка данных и подготовка признаков для моделирования

## Дисциплина
Анализ данных и искусственный интеллект

## Цель работы
Научиться готовить данные для алгоритмов машинного обучения, включая кодирование категориальных признаков, масштабирование числовых признаков, создание новых признаков и удаление нерелевантных данных.

## Предметная область
Датасет отражает поведение пользователей смартфонов, включая:
- количество времени, проведённого за телефоном и в соцсетях,
- рабочую продуктивность,
- сон и уровень стресса,
- использование приложений и потребление кофеина.

Цель анализа: подготовить данные для построения моделей машинного обучения, выявить ключевые признаки, влияющие на продуктивность и стресс.

## Исходные данные
- Файл: `data/raw/dataset_lab.csv`
- Строк: 50 000
- Признаков: 13 (числовые и категориальные)

## Используемые библиотеки
- pandas
- numpy
- scikit-learn

## Шаги выполнения
1. **Анализ категориальных признаков**  
   - `Gender`, `Occupation`, `Device_Type`  
   - Изучено распределение значений и баланс категорий  

2. **Кодирование категориальных признаков**  
   - **Ordinal Encoding** — присвоение числовых значений категориям  
   - **One-Hot Encoding** — создание бинарных признаков для каждой категории  

3. **Масштабирование числовых признаков**  
   - Использован `StandardScaler`  
   - Масштабированы признаки: `Age`, `Daily_Phone_Hours`, `Social_Media_Hours`, `Work_Productivity_Score`, `Sleep_Hours`, `Stress_Level`, `App_Usage_Count`, `Caffeine_Intake_Cups`, `Weekend_Screen_Time_Hours`

4. **Создание дополнительных признаков**  
   - `Total_Screen_Time` — сумма экранного времени в будни и выходные  
   - `Productivity_per_Sleep` — продуктивность на час сна

5. **Удаление нерелевантных признаков**  
   - `User_ID` — уникальный идентификатор, не влияющий на обучение модели  

6. **Подготовка финального датасета для модели**  
   - Сохранение двух вариантов:  
     - `dataset_final_ordinal.csv` — с Ordinal Encoding  
     - `dataset_final_onehot.csv` — с One-Hot Encoding
