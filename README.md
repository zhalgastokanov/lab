# Лабораторная работа №1: Исследовательский анализ данных (EDA) и подготовка данных

## Дисциплина
Анализ данных и искусственный интеллект

## Цель работы
Освоить полный цикл первичного анализа реального датасета, выявить зависимости между признаками, очистить данные и подготовить новые признаки для дальнейшего анализа.

## Предметная область
Датасет отражает поведение пользователей смартфонов, включая:
- количество времени, проведённого за телефоном и в соцсетях,
- рабочую продуктивность,
- сон и уровень стресса,
- использование приложений и потребление кофеина.

Цель анализа: изучить влияние поведения пользователя на продуктивность и стресс, выявить скрытые закономерности и подготовить данные для визуализации и моделирования.

## Исходные данные
- Файл: `data/raw/Smartphone_Usage_Productivity_Dataset_50000.csv`
- Строк: 50 000
- Признаков: 13 (числовые и категориальные)

## Используемые библиотеки
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn (при необходимости для предобработки)

## Шаги выполнения
1. **Описание предметной области и цели анализа**  
2. **Загрузка данных и первичный обзор**  
3. **Анализ пропусков и выбросов**  
4. **Очистка данных**  
5. **Feature Engineering**  
   - Создание новых признаков:
     - `Phone_to_Social_Ratio` — доля времени за соцсетями
     - `Sleep_Deficit` — недостаток сна относительно 8 часов
     - `Caffeine_per_App` — потребление кофеина на одно приложение
6. **Визуальный анализ зависимостей**
   - Корреляционная матрица
   - Scatter plot для ключевых зависимостей
   - Boxplot для категориальных признаков
7. **Формулировка аналитических выводов**
   - Экранное время снижает продуктивность
   - Недосып повышает стресс
   - Различия по полу и типу устройства
   - Caffeine_per_App связан со стрессом
   - Phone_to_Social_Ratio напрямую влияет на продуктивность
