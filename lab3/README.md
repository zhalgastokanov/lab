# Лабораторная работа №3: Построение первой модели машинного обучения

## Дисциплина
Анализ данных и искусственный интеллект

## Цель работы
Освоить базовый процесс обучения модели машинного обучения и оценки её качества.

## Исходные данные
- Подготовленный датасет из лабораторной работы №2 (`dataset_final_onehot.csv`).

## Задания
1. **Выбрать целевую переменную и обосновать выбор**  
   - Например, `Work_Productivity_Score` — для предсказания продуктивности пользователей на основе их привычек и поведения.  

2. **Разделить данные на обучающую и тестовую выборки**  
   - Использовать `train_test_split` из `sklearn` с фиксированным `random_state` для повторяемости.  

3. **Построить базовую модель**  
   - Линейная регрессия (для числовой целевой переменной) или логистическая регрессия (для категориальной целевой переменной).  

4. **Выполнить обучение модели**  
   - Подгонка модели на обучающей выборке.  

5. **Рассчитать метрики качества**  
   - Для регрессии: MAE, MSE, R².  
   - Для классификации: точность (accuracy), F1-score, матрица ошибок.  

6. **Проанализировать ошибки модели**  
   - Построить графики остатков (residuals), scatter plot предсказанное vs реальное, гистограмму ошибок.  
   - Проверить наличие выбросов и систематических отклонений.  

7. **Сделать выводы о качестве модели**  
   - Оценить, насколько модель предсказывает целевую переменную.  
   - Сделать рекомендации по улучшению модели (добавление признаков, изменение алгоритма, удаление выбросов и т.д.).
